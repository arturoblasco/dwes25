Son clases opuestas a abstractas, ya que evitan que se pueda heredar una clase o m√©todo para sobreescribirlo. Se define mediante `final class NombreClase {`.

```php
<?php
  class Producto {
    private $codigo;

    public function getCodigo() : string {
        return $this->codigo;
    }

    final public function mostrarResumen() : string {
        return "Producto ".$this->codigo;
    }
  }

  // No podremos heredar de Microondas
  final class Microondas extends Producto {
    private $potencia;

    public function getPotencia() : int {
        return $this->potencia;
    }

    // No podemos implementar mostrarResumen()
  }
```
